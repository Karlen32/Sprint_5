# Проект «Доска»  

Проект для автоматизации тестирования платформы объявлений «Доска».  

## Описание  
Тестируется функциональность сайта:  
- Регистрация пользователей  
- Авторизация пользователей  
- Размещение объявлений  

## Поддерживаемые браузеры  

- Тесты запускаются и проверяются вбраузере:  
- `Google Chrome ` 

## Используемые технологии: 
- Python, Selenium WebDriver, pytest.  

## Структура проекта  
- `tests/` — папка с тестами  
- `locators.py` — файл с группами локаторов, сгруппированных по страницам  
- `conftest.py` — файл с фикстурами для запуска браузеров и подготовки данных  
- `.gitignore` — исключенные файлы и папки (например, виртуальное окружение, временные файлы)  

## Установка  
1. Клонировать репозиторий:  

- `git clone git@github.com:Karlen32/Sprint_5.git`

## Перейти в папку проекта:

- `cd Sprint_5`


## Установить зависимости:

- `pip install -r requirements.txt`

## Запуск тестов

- Тесты запускаются из корневой папки проекта командой:

- `pytest -v tests/...`


## Зависимости

- Список зависимостей для requirements.txt:

- `pytest>=8.0`
- `selenium>=5.0`
